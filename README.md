# TicTacToe
Tic-tac-toe is a paper-and-pencil game for two players, X(**RED**) and O(**Yellow**), who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

# Features
* Multiplayer
* Simplistic UI

# Getting Started
* In android studio goto file
* Select new > Import from version control
* Choose git / github (git option recommanded )
* Paste url to repository i.e https://github.com/abhinav78910/TicTacToe.git
* Click on clone.

## Get it on [Playstore](https://play.google.com/store/apps/details?id=com.apps.simpletictactoe)
